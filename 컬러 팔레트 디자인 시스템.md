## 컬러 사용 가이드

### Primary (`#4A7C59` 포레스트 그린)
- 헤더 배경, 주요 CTA 버튼, 섹션 구분 라인
- 브랜드의 핵심 색상으로 신뢰감·안정감 전달
- **사용 예시**: `background-color: var(--color-primary);`

### Secondary (`#6B8FBF` 소프트 블루)
- 보조 버튼, 태그/뱃지, 하이퍼링크 기본 색상
- Primary 초록과 보색 관계로 시각적 균형 형성
- **사용 예시**: `color: var(--color-secondary);`

### Accent (`#C1392B` 딥 레드)
- 알림 배지, 긴급 강조 문구, 특별 할인·포인트 표시
- 전체 면적의 5% 이하로 제한하여 강조 효과 극대화
- **사용 예시**: `border-left: 4px solid var(--color-accent);`

### Background
| 변수 | 용도 |
|------|------|
| `--color-bg-primary` | 메인 페이지 배경, 카드 배경 |
| `--color-bg-secondary` | 섹션 교차 배경, 사이드바 배경 |

### Text
| 변수 | 용도 |
|------|------|
| `--color-text-primary` | h1, h2, 강조 텍스트 |
| `--color-text-secondary` | 본문 p 태그 |
| `--color-text-muted` | 날짜, 캡션, placeholder |

### Interactive
| 변수 | 용도 |
|------|------|
| `--color-button-primary` | 주요 액션 버튼 (저장, 제출) |
| `--color-button-hover` | 버튼 :hover 상태 |
| `--color-link` | a 태그 기본 색상 |
| `--color-link-hover` | a 태그 :hover 색상 |

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경을 어둡게 반전 */
    --color-bg-primary:    #1A1F1C;  /* 딥 다크 그린 베이스 */
    --color-bg-secondary:  #232A25;  /* 약간 밝은 다크 */

    /* 텍스트를 밝게 반전 */
    --color-text-primary:  #F0EDE6;  /* 크림 흰색 */
    --color-text-secondary:#C8C4BC;  /* 라이트 그레이 */
    --color-text-muted:    #8A8880;  /* 미디엄 그레이 */

    /* Primary/Secondary는 채도 약간 낮춰 눈 부담 감소 */
    --color-primary:       #5A9469;
    --color-secondary:     #7BA0CC;

    /* 테두리 어둡게 */
    --color-border:        #3A5070;
    --color-border-light:  #7A6840;
  }
}
```

### 화면 크기별 색상 활용 팁

| 화면 크기 | 고려사항 |
|-----------|---------|
| **Mobile (xs~sm)** | 버튼 배경색에 충분한 대비 확보. `--color-button-primary`의 명도비 4.5:1 이상 유지 |
| **Tablet (md)** | 사이드바에 `--color-bg-secondary` 활용하여 콘텐츠 영역 구분 |
| **Desktop (lg~)** | 히어로 섹션 배경에 `--color-primary` 그라디언트 적용 가능 |

### 접근성 (WCAG AA 기준)
- `--color-text-primary` (`#1E2B3C`) on `--color-bg-primary` (`#F5F0E8`) → 대비율 **약 11:1** (통과)
- `--color-text-secondary` (`#4A4A4A`) on `--color-bg-primary` (`#F5F0E8`) → 대비율 **약 7:1** (통과)
- Accent 빨간 버튼 사용 시 흰색 텍스트 권장 → `#FFFFFF` on `#C1392B` → 대비율 **약 5.8:1** (통과)
